Getting Started: Running Seltzer

Running Seltzer is very simple. There are 4 things you need:
  • Seltzer Core
  • A 1.7+ Java installation
  • A Google Chrome installation
  • The Chrome web driver (here)

To prepare, make your Seltzer directory. It's recommended it be somewhere easy to get to with your shell/command line. Take your copy of Seltzer Core, probably named something like seltzer-core-x.y.z-jar-with-dependencies.jar, where x.y.z represents the version you've downloaded/compiled. For brevity, the rest of this document will call it seltzer-core.jar.Also, make a subfolder called web_drivers and put your Chrome web driver in there.

In order to actually start Seltzer, navigate a shell/command prompt to your Seltzer directory. Then you simply run it with the command java -jar seltzer-core.jar -Dseltzer.path=/absolute/path/to/seltzer/dir. The seltzer.path property that is set in the command is absolutely required (unless you set it in your config file), and it must be an absolute path. This is the directory that Seltzer will place the new browser profiles in, as well as look for the web driver.
