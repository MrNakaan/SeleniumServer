1.x Apple

Aspen Soda on Wikipedia.
Release on GitHub
Features
  • The following commands:
    • Click
    • Go to URL
    • Back
    • Forward
    • Count items
    • Fill field
    • Submit form
    • Delete elements from page
    • Get cookie(s)
    • Get cookie file
    • Get current URL
    • Read attributes from elements
    • Read text out of elements
    • Send keys
    • And waiting for any of a ton of conditions that Selenium can wait for
  • Support for all of the By selectors that Selenium supports (though deleting elements is limited to xpath for now)
  • Responses from those commands include any response data and a success flag
  • Chain commands, which allow you to group any number of commands into a single big one to send off
  • OSGi compatibility for command/response and utility modules
  • Multiple sessions, allowing many programs to use the same Seltzer server without interfering with each other
  • Automatic cleanup of unused/abandoned sessions
  • Headless mode for browser "windows"
  • A utility for communicating with Seltzer servers that takes care of the "gotcha" scenarios for you
