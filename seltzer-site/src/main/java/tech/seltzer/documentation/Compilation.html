Getting Started: Compiling Seltzer

Seltzer is compiled using Apache Maven. Compilation is a very simple process. Using a shell/command line, navigate into the seltzer-parent directory and run mvn install -Dseltzer.path=/absolute/path/to/seltzer/dir. You can skip setting seltzer.path if you tell Maven not to run unit tests. Skipping unit tests isn't recommended, but the unit tests can take quite a long time (10+ minutes) so the option is good to have.
