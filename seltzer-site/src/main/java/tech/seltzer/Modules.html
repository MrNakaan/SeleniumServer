Modules

Below is a list of all the modules within the Seltzer project.
  • Parent
    Type: Aggregate
    Path: /seltzer-parent
    Description: This is simply the parent element to all the Java modules, and should not be used as a dependency.
    Documentation: Coming soon...
  • Core
    Type: Standalone Application
    Path: /seltzer-parent/seltzer-core
    Description: Core is the Seltzer server application. It is run separately from any applications that include any of the library modules listed on this page.
    Documentation: Coming soon...
  • Balancer
    Type: Standalone Application
    Path: /seltzer-parent/seltzer-balancer
    Description: Coming in Seltzer 2.0
    Balancer is a load balancing application for managing multiple Seltzer servers.
    Documentation: Coming soon...
  • Command/Response
    Type: Library
    Path: /seltzer-parent/seltzer-cr
    Description: Command/Response is the module responsible for providing a common interface between Seltzer Core and any application needing to communicate with it. Command/Response provides the command data classes that are sent to a Seltzer server, as well as the response classes that the server will return. Along with this are enumerations used to provide important metadata and exceptions.
    Documentation: Coming soon...
  • Util
    Type: Library
    Path: /seltzer-parent/seltzer-util
    Description: Util is exactly what the name implies, a module providing utility classes and methods. Currently available are a send/receive implementation and a [deprecated] command data factory. A more fluent command builder is planned for a future release.
    Documentation: Coming soon...
  • Server Utils
    Type: Library
    Path: /seltzer-parent/seltzer-server-util
    Description: Coming in Seltzer 2.0
    The server utils module contains code that is shared by both Seltzer Core and Seltzer Balancer.
    Documentation: Coming soon...
  • Site
    Type: Website
    Path: /seltzer-site
    Description: This module is simply the site that you're currently viewing.
    Documentation: Coming soon...
